## Tech stack

| Layer | Technology |
|---|---|
| Runtime | Node.js |
| Framework | Svelte 5 (Runes) |
| Meta-framework | SvelteKit 2 (SSG) |
| Styling | Tailwind CSS 4 |
| Localization | svelte-i18n |
| Animation | anime.js |
| Distribution | GitHub Pages / Cloudflare Pages |

## Key features

<CardGroup cols={2}>
  <Card title="Bento-grid layout" icon="grid-2">
    Responsive glassmorphism card grid adapts from single-column mobile to a 4-column desktop layout
  </Card>
  <Card title="Dynamic accent colors" icon="palette">
    Each visitor gets a randomly assigned accent color persisted in a cookie, injected server-side to prevent FOUC
  </Card>
  <Card title="Internationalization" icon="language">
    Full UI translations in English, Italian, German, and Spanish via svelte-i18n with cookie-based locale persistence
  </Card>
  <Card title="GitHub integration" icon="github">
    Live project archive fetched from the GitHub REST API with in-memory caching and language breakdown stats
  </Card>
  <Card title="Stats.fm widget" icon="music">
    Real-time music activity card powered by the Stats.fm API showing current and recent listening
  </Card>
  <Card title="Static output" icon="bolt">
    SvelteKit SSG adapter produces a fully static site deployable to any CDN with zero server runtime
  </Card>
</CardGroup>

## Pages

The site has four public routes:

- **`/`** — Home: bio card, about section, music widget, live clock
- **`/projects`** — Archive: GitHub repos with language breakdown chart
- **`/contact`** — Contact hub: social links and email
- **`/uses`** — Setup: curated list of hardware and software
